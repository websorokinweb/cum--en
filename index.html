About CUM

CUM coin is a cryptocurrency in the Smart Chain network(BEP20).

The basic idea of the project is to create a green, fast and secure currency for making payments. This project is backed and designed for a North American audience (Canada, USA, Mexico).

The idea of these three countries served as the basis for the coin's name — CUM. These countries have chosen because of their positive level of economic development. However, this cryptocurrency can be used everywhere in the world without any restrictions.
